# Travel-App

## About Project : 
This is last project of udacity's course in which students build a travel app and pull basic data from an API.The project will include a simple form where you enter the location you are traveling to and the date you are leaving. If the trip is within a week, you will get the current weather forecast. If the trip is in the future, you will get a predicted forecast.
